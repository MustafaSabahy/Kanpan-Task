# Screenshots Folder

Place your app screenshots here. The images will be automatically displayed in the main README.md file.

## Required Screenshots

- `cover.png` - Main cover image for the repository
- `home_screen_light.png` - Home screen in light mode
- `home_screen_dark.png` - Home screen in dark mode
- `task_details.png` - Task details screen
- `task_details_dark.png` - Task details in dark mode
- `statistics.png` - Statistics screen
- `statistics_dark.png` - Statistics in dark mode
- `history.png` - History screen
- `history_dark.png` - History in dark mode
- `splash_screen.png` - Splash screen
- `language_selection.png` - Language selection menu

## How to Take Screenshots

### iOS Simulator
1. Open the app in iOS Simulator
2. Press `Cmd + S` to take a screenshot
3. Screenshots are saved to Desktop by default
4. Move them to this folder

### Android Emulator
1. Open the app in Android Emulator
2. Use the screenshot button in the emulator toolbar
3. Save images to this folder
